# TIL Book

Today I Learned. 그날그날의 배운 내용들을 간결하게 정리하여 모아두었습니다.

### How to run in local

```bash
$ gitbook install  # Install plugins
$ gitbook serve  # Run local server
```

More informations about gitbook is [here](https://toolchain.gitbook.com/setup.html).
